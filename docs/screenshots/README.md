# 截图说明

请启动看板后截图并放置到本目录：

1. `dashboard-overview.png`  — 总览面板（含流程管线 + 图表 + Agent健康）
2. `dashboard-kanban.png`    — 任务看板（Kanban列视图）
3. `dashboard-history.png`   — 工作记录（流转日志展开状态）
4. `dashboard-timeline.png`  — 时间线视图
5. `dashboard-models.png`    — 模型配置面板

截图方法：
- macOS: `Cmd+Shift+4` 框选区域
- 或运行: `screencapture -x -T 2 docs/screenshots/dashboard-overview.png`
